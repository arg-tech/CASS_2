The overall similarity, simply, provides an average of the text and graph similairy measures.

The text similarity measure provides the overall mean for text nodes in an AIF graph (l-nodes and i-nodes if both are present). Connections are selected through a matrix approach meaning all nodes have a corresponding connected node. Fuzzy string matching is then used to provide a normalised text score for the full matrix. Matches are then selected by similarity score until no nodes are left to be matched.

The graph similarity measure provides the overall mean for all component parts in an AIF diagram (e.g. i-nodes, i-nodes and s-nodes combined, i-nodes, l-nodes, and s-nodes etc.). Graphs are created from through NetworkX and the Graph Edit Distance calucalted for set, creating a distance matrix. The minimum value from the distance matrix is then selected (except 0). The mean is then taken for all calculations to give a similarity score.
